# sss

[Small SAT solvers](https://web.archive.org/web/20201109101535/http://www.cse.chalmers.se/~algehed/blogpostsHTML/SAT.html)
written in several programming languages for educational purposes.

## Run examples

### APL

Copy-paste [sss.apl](./apl/sss.apl) into [TryAPL](https://tryapl.org/).

### BQN

Copy-paste [sss.bqn](./bqn/sss.bqn) into [BQNPad](https://bqnpad.mechanize.systems/).

### Clojure

```shell
cd clojure
clojure -m sss
```

### Go

```shell
cd go
go run sss.go
```

### Java

```shell
cd java
java Sss.java
```
